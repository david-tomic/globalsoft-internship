1. _Read through htmlToLearn_, navigate it as a roadmap / guidelines. (pick the topics that you are not familiar with and investigate)

2. _Read through cssToLearn_, navigate it as a roadmap / guidelines. (pick the topics that you are not familiar with and investigate). Note while html is alot more barebones, css packs alot more stuff and has more quirks. Main thing to internalize is box model, positions, displaying, responsive design, basically how to layout stuff.

3. Once you think you are ready / bored from reading all the articles, have a go at implementing https://www.frontendmentor.io/challenges/product-preview-card-component-GO7UmttRfa - files are provided in 01/exercise

Note: disregard mumbo jumbo they provide, just focus on styling / layout, try to follow what you've read, and use the design/images they provide. It does not have to be pixel perfect.

Optional: if you dislike this feel free to implement some other card-like layout, with some images, responsivness that utilizes media query for displaying differently in mobile, uses some other fonts, etc.

4. If you find yourself struggling / wanting to practice more feel free to browse through https://www.frontendmentor.io/challenges?sort=startedCount%7Cdesc
